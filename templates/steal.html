Steal this Website

Intro

This website is a portfolio. I built it in a day using free tools, open-source software, and about fifteen dollars and you can even shave a few dollars off that if you want.

I’ve been thinking about this guide for a while now, mainly of what the structure and scope would be. I decided that this guide will follow a six-month period in late 2021-22 when I tried to answer the question "how do you make a website with the Python programing language?”

That’s a big question. No one unprompted would ask that question. I won’t go into exactly why I asked myself that question, suffice to say it happened and it led me down an interesting path. I learned A LOT along that path. Getting from setting up a development environment to a live website the first time was incredibly rewarding and I want to share that journey the best with anyone hoping to do the same.

Who is this Guide For?

The most obvious answer is “someone who wants to build a website”.

However a useful answer would be closer to someone willing to learn how to build a website while getting your hands dirty. There are companies out there who have simplified the creation of a website to the level of creating a PowerPoint presentation. That’s an incredible achievement because everyone who wants a website, deserves a website. The barrier of entry shouldn’t be a six month self-induced crash course. That should just be for fun.

Another candidate would be anyone looking for an entry-point into Python. I’m not a developer, more of an enthusiast. An aspiring tinkerer. However, a simple website using Flask, which is a Python framework - or, a collection of pre-written code (Python in our case) that provides the tools and structure to help develop a kind of software program (in our case, websites) - was an interesting project that allowed me to practice my Python. You’ll learn how to “setup a programing environment”, how to use “Git”, basic Python, “package management”, and more.

Lastly, and this might be a stretch, any technology-curious person without the traditional technology background looking for a broad exposure to the technology world. I’m obviously “self-taught”, or else why would I ask that original question of “how do you make a website with the Python programing language?” There’s a million and one guides online explaining how to build a website, maybe this one is the one that finally gets you over the hump - a nice thought.

Also, any comments or suggestions and most importantly corrections, please feel free to reach out! This is a learning experience for me just as much as any hypothetical reader, let’s talk.
